
外出經費申請輔助系統（Internal v1.0 正式版）

使用方式：
1. 請先下載 xlsx.full.min.js
   https://cdn.jsdelivr.net/npm/xlsx/dist/xlsx.full.min.js

2. 與 index.html、app.js 放在同一資料夾

3. 直接以瀏覽器開啟 index.html

4. 匯入 104HR 外出申請單 Excel（匯出檔）

說明：
- 起點畫面顯示為「台中市」，實際路線仍自台中西屯出發
- 日期格式固定為 YYYY/MM/DD
- Google 路線為「去 → 多點拜訪 → 回台中」

此工具不連接公司系統、不存資料，僅做轉換與整理，用於輔助經費申請。
